HTMLClass

Este é um simples projeto na qual desenvolvi para se trabalhar com tags HTML via PHP.
Simplesmente atraves de uma instancia da classe tag, voce pode manipular os elementos HTML.

Fazer carregamento automático de arquivos .css e .js. Entretanto este código não é 100% aplicado a 
qualquer projeto. Visto que desenvolvi baseado em minhas necessidades. Talvez em algum outro projeto
aconteça a necessidade de se implementar alguma coisa a mais.
